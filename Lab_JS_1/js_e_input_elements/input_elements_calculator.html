Javascript Input Elements Exercises

HTML input elements are primarily used in HTML forms. Form data is submitted to the server for processing. In the sample form below a php script on a server processes the submitted data.

Input elements are also a general mechanism for collecting input from users. This input can be accessed by Javascript for client-side processing.

HTML5 defines a large number of input types, listed here. Creating an input element of a particular type harnesses the HTML5 interpreter's capability for input validation.

Exercise: Calculator [EX02]

The following html elements are defined:

  • two input fields
  • four buttons: "Add", "Subtract", "Multiply" and "Divide"
  • an empty span element with id answer

Your task is to write Javascript code that implements a calculator. When any of the buttons is clicked, the requested operation should be performed on the numbers in the two input fields. The result should be displayed in the element with id answer.

Place your code in the file ex.js and create a script element to include ex.js in this HTML file.

Tips:

  • The value entered by the user into an input field is stored in the element object property value (not innerHTML).
  • After reading the value from the input element, pass it into the function parseInt() to convert it from string type to number type, otherwise you might get some surprising results after using the arithmetic operators.
  • In order to access the button and input elements, you can use the document.getElementById() method, for which you will have to define some ids on the elements.
